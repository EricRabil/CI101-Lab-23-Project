Card Gallery Layout

This layout has a nav bar above the header image and is responsive, allowing for equal width "cards" with content like images, headline and text. How many cards are in each row is dependent upon the width of the viewport. Be careful, the height is dependent upon the length of the content.

Two Equal Columns Layout

This layout has a header followed by navigation bar. The content is in two equal width columns.

Three Un-Equal Columns Layout

This layout has a header with a vertical navigation bar inside of it. The content is in three un-equal width columns. There is a narrow column, a wide center column and another narrow column. There is a fine line separating them.
